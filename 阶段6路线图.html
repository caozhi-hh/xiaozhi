Direction B — Second Priority

记忆系统

P1
🧠
三层记忆架构
短期记忆管对话上下文，中期记忆学用户偏好，长期记忆积累知识
  • 短期记忆：对话上下文自动管理（已有基础，优化 token 控制）
  • 中期记忆：用户偏好学习（喜欢的风格、常用模型、常用工具）
  • 长期记忆：跨对话知识积累（用户信息、项目背景、习惯）
  • 数据库新增 user_memory 表
  • 自动摘要 + 关键信息提取 → 存入记忆
  • 每次对话自动注入相关记忆到 System Prompt
DeepAgents MemoryMiddleware LangGraph Checkpoint SQLite
🔍
记忆检索与使用
小智每次回答前自动检索相关记忆，越用越懂你
  • 记忆分类：个人信息 / 偏好设置 / 项目背景 / 交互历史
  • 对话开始时加载用户画像
  • 用户说"记住我喜欢简洁回答" → 自动存入偏好
  • 前端记忆管理页面（查看/删除/编辑）

RAG 知识库

P2
📖
文档处理 + 向量化

  • Embedding 模型：通义 text-embedding-v3
  • 向量存储：Chroma（本地轻量，无需额外服务）

LangChain Loaders text-embedding-v3 ChromaDB

  • 回答时标注引用来源和文档片段
  • 作为 DeepAgent 工具注册：search_knowledge_base
